# CanvasApp

CanvasApp is a collaborative drawing web application that allows multiple users to draw in real time on the same canvas. Users can customize their drawing experience with a variety of features, including color selection, undo functionality, and theme switching. Advanced features such as saving drawings and multi-user capabilities enhance the experience.

## Features

- **Real-Time Drawing**: Multiple users can draw simultaneously on a shared canvas.
- **Color Selection**: Choose from a wide range of colors beyond the default palette.
- **Undo Functionality**: Easily revert the last action.
- **Reset Canvas**: Clear the entire canvas with a single click.
- **Light/Dark Mode**: Switch between light and dark themes for a comfortable experience.
- **Save Drawings** (Upcoming): Users will be able to save their artwork.
- **Multi-User Support** (Upcoming): Enable collaborative drawing sessions with multiple participants.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mikeyfera1/CanvasApp.git
   ```
2. Navigate to the project directory:
   ```bash
   cd CanvasApp
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   node server.js
   ```

## Usage

1. Open the application in your browser.
2. Select a color and start drawing.
3. Use the undo button to revert the last stroke.
4. Reset the canvas if needed.
5. Switch between light and dark mode for better visibility.
